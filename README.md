# gro2pdb.py

## 概要
Program to convert .gro to .pdb file


## 使用方法
```sh
$ gro2pdb.py [-h] -i INPUT.gro -o OUTPUT.pdb [-O]
```

options:

* `-h`, `--help`
	: show this help message and exit
* `-i INPUT.gro`
	: source .gro file
* `-o OUTPUT.pdb`
	: output .pdb file
* `-O`
	: overwrite forcibly



## 動作要件
* Python3
	* parmed
